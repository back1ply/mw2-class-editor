# mw2-class-editor
Class editor for Call of Duty Modern Warfare 2 (2009) Multiplayer, works with IW4X.

## Project Structure

- `index.html`: Main entry point for the web application.
- `js/`, `css/`: Core application logic and styling.
- `img/`: Game assets and icons.
- `tools/`: Maintenance and utility scripts.
  - `tools/python/`: Scripts for IWI conversion and asset extraction from game files.
  - `tools/js/`: Scripts for downloading assets and auditing missing files.
  - `tools/debug/`: Exploration and one-off debug scripts.
- `tasks/`: Internal tracking and lessons learned.
